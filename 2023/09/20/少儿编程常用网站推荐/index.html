少儿编程常用网站推荐

发表于 2023-09-20 阅读次数： 本文字数： 1.7k 阅读时长 ≈ 3 分钟

第一章

1 引言

少儿编程是通过编程游戏启蒙、可视化图形编程等课程，培养学生的计算思维和创新解难能力的课程。针对6-18岁的少年儿童开展的编程教育，最常见的形式是线上和线下模式相结合的课外培训。
根据先易后难的学习进程，少儿编程教学可以大致分为两类。一类是Scratch或是仿Scratch的图形化编程教学，以培养兴趣、锻炼思维为主，趣味性较强。在这里，孩子们可以创造属于自己的动画、故事、音乐和游戏，这个过程就像搭积木一样简单。此外，还有机器人编程，也就是搭建机器人，通过运行程序让它动起来，着重培养孩子的动手能力。另一类是基于Python、C++等高级编程语言的计算机编程教学，目标往往是参加信息学奥赛等科技品牌赛事，或为后续的专业学习和职业技能打下基础。在这里，孩子们可以熟悉编程原理，执行代码操作，适合有一定数学基础、英语基础和逻辑思维的孩子。

2 少儿编程常用网站推荐

2.1 Code.org

推荐指数：★★★★★
适宜年龄：4-18
官网网址：http://www.code.org
简介：Code.org 是全球最大型的计算机科学和少儿编程布道非盈利组织，旨在通过免费优质的在线课程资源，让全世界的孩子们都能享受到最前沿的计算机教育。

2.2 Scratch

推荐指数：★★★★★
适宜年龄：5-14
官网网址：http://scratch.mit.edu
简介：Scratch 是目前最流行的少儿编程软件，提供图形化、积木式的编程界面，让孩子能轻松上手学习编程知识。其官网的作品展示非常丰富，有来自全世界孩子们的编程作品。

2.3 少儿编程学院

推荐指数：★★★★
适宜年龄：6~
官网网址：http://edu.shaoerbc.org
简介：少儿编程学院，是由「中国少儿编程教育网」推出的少儿编程教育MOOC学院，通过公益免费的形式，让知名教师入驻学院，提供公开课、录播课、直播课等优质的免费课程。

2.4 编程猫社区

推荐指数：★★★★
适宜年龄：7~16
官网网址：https://shequ.codemao.cn/course
简介：一个免费的源码编辑器。网页里有一些不错的免费课程，像scratch、python都有，可以做一个小小的启蒙。

2.5 Tynker

推荐指数：★★★★★
适宜年龄：7~
官网网址：https://www.tynker.com/
简介：该网站是美国一家极具影响力的少儿编程在线学网站。其创始人的理念在于：教孩子怎样像程序员一样思考，而不是教他们如何写出长串代码。
为了激发孩子对编程的天然热爱，该网站采用游戏+社交的方式让孩子学习编程。基础部分依然采用类似乐高积木的模块帮助孩子对游戏进行编程，而想要进一步学习的孩子，可以切换到Javascript或Python语言，查看脚本。

2.6 Code Combat

推荐指数：★★★★★
适宜年龄：10~
官网网址：https://codecombat.cn/
简介：Code Combat是一个编程游戏网站，整个网站就是一个网页游戏，孩子们需要通过编写代码，让游戏主角完成任务通关，真正将玩和学结合起来。孩子可以通过编写代码程序，控制左侧的主角，完成该关卡的任务，即可通关。该平台目前支持Python、Js等主流编程语言。

2.7 Light Bot

推荐指数：★★★★
适宜年龄：7~
官网网址：http://www.lightbot.com
简介：Light Bot是一个面向幼龄小孩，提供编程思维启蒙的游戏，孩子通过设计指令，让游戏中的小人通过路径把灯点亮。

2.8 ICode

推荐指数：★★★★★
适宜年龄：6~18
官网网址：https://www.icode.org.cn/
简介：“ICode国际青少年编程竞赛”是由ICode在全球举办的国际比赛。ICode旨在全球范围内推广编程教育。通过区域、国家和国际多形式的赛事，为全球学生创造一个免费、有趣、有效的学习和交流平台。

3 少儿编程的好处

少儿编程教育的好处在于培养孩子的逻辑思维能力和创新解难能力，同时也可以帮助他们提高数学、英语等学科的成绩。此外，少儿编程教育还可以培养孩子的团队协作能力和沟通能力，让他们在科技领域中获得更多的机会和发展。
